# Recursos

## Scala
### Libros
- [Programming in Scala, Third Edition](https://www.safaribooksonline.com/library/view/programming-in-scala/9780981531687/) El libro que utilicé para explicaros Scala, lo recomiendo especialmente si realmente quereis aprender del lenguaje, está muy orientado en el aspecto funcional del lenguaje pero los primeros capítulos están muy bien
- [Scala Cookbook](https://www.safaribooksonline.com/library/view/scala-cookbook/9781449340292/) Un libro de recetas. Está muy bien porque te enseña Scala pero haciendo cosas en concreto, si alguna vez quereis ver algo tipo "como hago x en Scala" este libro puede tener probablemente la solución

### Videos
- [Learn by Example: Scala](https://www.safaribooksonline.com/videos/learn-by-example/9781788993005) No lo he visto pero tiene buena pinta, autodescriptivo
- [Scala for the Impatient](https://www.safaribooksonline.com/videos/scala-for-the/9780134510613) Este si lo he visto y lo recomiendo bastante

### Páginas
- [Learn Scala in Y minutes](https://learnxinyminutes.com/docs/scala/) Basicamente un mini tutorial para la sintaxis del lenguaje y cosas esenciales
